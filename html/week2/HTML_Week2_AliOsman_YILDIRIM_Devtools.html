Crome DevTools

Crome DevTools ; bir HTML sayfasinin görümünü anlik olarak degistirmemizi ve bu degisiklikleri anlik olarak göruntulememizi saglayan bir programdir.
Ayrica bu program sayesinde ayni anda hazirladimiz sayfanin ceptelefonu, tablet veya bilgisayarda nasil görünecegini görmemize imkan saglar

Bir HTML sayfasinda Crome DevTools nasil acilir?

  • Crome DevTools acmak icin önecelikle kullanmak istedigimiz HTML sayfasini Google Crome ile acariz
  • Actigimiz sayfanin her hangi bir yerinde mause sag tiklayarak en alta bulunan incele butonuna basmamiz Crome DevTools un acilmasi icin yeterli olacaktir.
  • Yada ctrl+shift+I kisayol tusulari kullanilarakta acilabilir.

DevTools Paneli ilk olarak karsimiza Elements paneli ile acilir. Burda bulunan HTML karekterleri DOM agacida görülür. Dom agacinda gezinmek icin ok tuslari kullanila bilir. Actigimiz Web sayfasinda bir alanin Dom agacinda bulunan ilgili bölümüne gelmek icin simgesine tiklanir. Daha sonra Web sayfsinda istenilen bölüme gelerek mausun sol tusuna basildiginda ilgili bölümün HTML kodlari acilir.

Hazirladigimiz Web Sayfasinin Farkli Cihazlarda Nasil Görünür?

Crome DevTools bizlere hazirladigimiz Web sayfasinin Ceptelefonu,Tablet veya Bilgisayarda nasil görünecegini bize gösterme imkanini saglar. Bunun icin butonuna basilir ve asagidaki resimde yer alan cihaz secimi, sayfayi yan yatirma gibi özellikler karsimiza cikar.


DOM Agacinda klavye ile nasil gezilir?

Dom Agacinda daha öncede söyledigimiz gibi klavye ile gezmek mümkündür.

  • Dom Agaci icerisinde Yukari ve Asagi hareketi icin Yukari-Asagi yön tuslari kullanilir.
  • Sag yön tusu bir listeyi genisletir. Sol yön tusu ise bir listeyi daraltir.
  • Enter ise attribut eklemeye yarar.
  • Backspace veya Delete tusuna basarak bir tagi silmek mümkündür.

Scroll into view Ne ise yarar?

DOM alaninda islem yaparken bazen görüntülenen sayfada ilgili tagdan farkli bir noktayi göruntuleye biliriz. Bu durumda Scroll into view secilerek kisa yoldan calistigimiz DOM alanini görüntülenebilir.

DOM Icerisinde Arama Yapma

  • Öncelikle imlecimizi arama yapmak istedegimiz bölüme yerlestiririz.
  • Daha sonra Ctrl+F tusuna basarak arama alani acilir.
  • Acilan bolume aranmak istenen kelime yazilir

DOM ile icerik Nasil Düzenlenir?

  • Degistirilmek istenen bölümün üzerine gelinerek incele dügmesine basilir.
  • DOM Agacindaki duzeltilecek alananin icine mause ile cift tiklanir.
  • Istenilen degisiklik yapilarak enter tusuna basilir,

Attributleri Düzenleme

  • Degistirilmek istenen bölümün üzerine gelinerek incele dügmesine basilir.
  • Degistirilmek istenilen attributenin yaninda cift tiklanir.
  • Istenilen degisiliklik (örnegin;p taginin yanina style="background-color:gold" eklenir.)yapilarak enter tusuna basilir.

Attributlerin Tipini Degistirmek

  • Degistirilmek istenen bölümün üzerine gelinerek incele dügmesine basilir.
  • Degistirmek istenen attribut cift tiklanir
  • Istenen yeni tur attribut yazilip enter tusuna basilir.

DOM Ögelerinin Sirasini Degistirmek

  • Siralamasi Degistirmek istenen ögenin üstine gelerek incele dügmesine basilir.
  • Üzerine basili tutularak istenilen siraya cikarilir.

Bir Ögeyi Gizlemek

  • Gizlenmek istenilen ögenin üzerine gelerek incele denilir
  • H tusuna basilarak istenilen öge gizlenir.
  • Tekrar Htusuna basilirsa gizlenen nesne görünür hale gelir.

Bir Ögeyi Silmek

  • Silinmek istenen ögenin uzerine gelerek incele tusuna basilir.
  • Delete tusuna basilrak öge silinir.
  • CTRL+Zile yapilan degisiklik geri alinabilir.

CSS veya JS Yolunu Kopyalama

  • CSS veya JS yolunu kopyalamak istedigimiz ögenin uzerine gelerek incele butonuna basilir.
  • Elements kisminda mausenin sag tusuna basilir.
  • Kopyala kismindan ögenin styleni veya js yolunu kopyalaya biliriz.
